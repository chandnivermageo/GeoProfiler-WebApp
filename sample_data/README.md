# GeoProfiler Sample Data

This folder contains sample datasets for testing GeoProfiler.

## Files

### DEM
- sample_dem.tif

### River Profiles
- rivers.json
- rivers.shp (+ supporting shapefile components)

### Cross Sections
- cross_lines.shp (+ supporting shapefile components)

## Quick Start

1. Upload sample_dem.tif
2. Upload rivers.json (or the river shapefile)
3. Select the profile layer
4. Generate line or swath profiles
